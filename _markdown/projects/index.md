## [SLIM](/projects/SLIM/)
<p>A C++ library that provides a programming interface for easily creating, and managing a server database of user credentials in a cryptographically secure way. The library handles salting and hashing of passwords, login validation, and the management of temporary login tokens such as web cookies.
</p>

Forums and documentation

  • Arch Wiki - Best wiki for Unix and GNU software/tools, focused specifically for Arch Linux, but most information works on other OS's too
  • W3Schools
  • Stack Overflow
  • Super User
  • Man Pages - alternatively, you can just use the 'man' command in a Linux terminal
  • MDN Web Docs - Detailed documentation and guides for different parts of the web, run by Mozilla

Software

  • Mozilla Thunderbird - amazing email client that allows you to escape the grasp of google and microsoft mail clients, even if you're still bound to using them as your mail provider.
  • Calibre - for viewing, managing, and sharing ebooks
  • Mullvad VPN - doesn't track your data like NordVPN and others
  • Linux operating system - Not for everyone, but the most free your computer will ever be. Mint is a good flavor for beginners. Currently I'm using Arch, which is catered towards more advanced users.

Textbooks / Reference Manuals

  • The C Programming Language
    (Wikipedia)

  • UNIX Network Programming, 3rd edition, vol 1 - An essential guide book for learning how to make programs that communicate across a network. If you're working outside of a POSIX environment, the code itself won't work, but general concepts still apply.
    (Wikipedia)

  • Introduction to Algorithms - "CLRS" - Useful reference for Data Stuctures
    (Wikipedia)
